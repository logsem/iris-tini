An overview of the Coq formalization

The development is known to compile with:

  • coq: version 8.11.1
  • coq-iris : version dev.2020-04-09.2.962637df
  • coq-stdpp : version dev.2020-04-03.1.eeef6250
  • coq-autosubst
  • coq-iris-string-ident

To obtain the development versions, you have to add the following Coq opam repository:

opam repo add iris-dev https://gitlab.mpi-sws.org/iris/opam.git

Once you got opam set up, run opam update followed by make build-dep to install the right versions of the dependencies.

Run make -jN to build the full development, where N is the number of your CPU cores.

Modal Weakest Precondition Theory

The Modal Weakest Precondition theory is in our Coq formalization called If-Convergent (IC).

  • The general definition of the IC predicate (Definition 3.4) is defined in IC.v.
  • All general derived proof rules for program reasoning are defined in IC_lifting.v.
  • The adequacy statement is stated and proved in IC_adequacy.v.
  • The example predicate (Example 3.1) is defined in IC_fupd.v.
  • The unary predicate (Example 3.2) used in our logical-relations model is defined in IC_step_fupd.v.
  • The binary predicate (Example 3.5) used in our logical-relations mode is defined in IC_left.v; the inner predicate in IC_right.v
  • The binary predicate used in Section 5 is defined in IC_logrel_fupd.v
  • Lemmas about the interactions between the unary and binary predicates (e.g. Lemma 3.7 and Lemma 5.1) are found in ni_logrel_lemmas.v and ni_logrel_fupd_lemmas.v

Noninterference Development

  • Our language of study is defined in lang.v.
  • The types and type system are defined in types.v and typing.v, respectively.
  • The unary logical relation is defined in logrel_unary.v and its fundamental theorem is stated and proved in fundamental_unary.v.
  • The binary logical relation is defined in logrel_binary.v and its fundamental theorem is stated and proved in fundamental_binary.v.
  • Termination-insensitive noninterference is proved in noninterference.v.
  • The introductory examples are found in arith.v, refs.v, and refs_implicit.v.
  • The example from Section 5.1 is found in report.v.
  • The examples from Section 5.2 are found in value_dependent.v and value_dependent_pack.v.
  • The example from Section 5.3 is found in compute_with_cache.v.
  • The parametricity theorems from Section 5.4 are shown in parametricity.v.
